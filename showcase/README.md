# Design Showcase

This directory preserves two standalone visual explorations for the repository homepage and Qwopus3.6 learning experience.

| Version | Focus | File |
|---|---|---|
| V1 | GitHub-inspired resource portal with a compact training-first layout | [`README_VISUAL_PREVIEW.html`](README_VISUAL_PREVIEW.html) |
| V2 | Qwopus3.6 product family plus fine-tuning curriculum, scripts, architecture diagrams, and local deployment | [`QWOPUS_META_STYLE_V2.html`](QWOPUS_META_STYLE_V2.html) |

GitHub displays standalone HTML as source rather than executing it. To view the pages as designed, clone the repository and open either HTML file locally, or serve the repository with a local static-file server.

The production-facing GitHub repository homepage remains [`../README.md`](../README.md), which uses GitHub-compatible Markdown and sanitized HTML.
